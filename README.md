# Matchy - Farcaster Mini App

A location-based matching app for people at events.

## Features

- Find matches nearby at events
- Filter by categories and radius
- View event listings
- Edit profile with social links

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Farcaster Mini App SDK

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000
