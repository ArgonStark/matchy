# Image Files Needed

Please add the following image files to the `public` folder:

1. **logo.png** - The Matchy logo (teal background with location pin and network icon)
   - Recommended size: 200x200px or 1024x1024px
   - Format: PNG with transparency or solid teal background
   - Used in: TopBar component, Farcaster manifest

2. **get-started.png** - The landing page illustration (isometric pixel art scene)
   - Recommended size: 400x400px or larger
   - Format: PNG
   - Used in: Landing page (/)

3. **preview.png** - Preview image for Farcaster frame metadata
   - Recommended size: 1200x630px (3:2 ratio)
   - Format: PNG
   - Used in: Farcaster frame metadata

Once you add these files, the app will automatically use them!
